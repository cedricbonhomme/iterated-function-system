Iterated Function Systems with OCaml
====================================

In your OCaml interpreter type:

```ocaml
# #use "ifs_fractals.ml";;
# trace barnsley 200000;;
```

![Barnsley Fern](example/barnsley.png "Barnsley Fern")

This work was carried out during a functional programming course.
